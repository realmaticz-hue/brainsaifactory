2️⃣ Project Graph Memory (Critical)
System must maintain a live project graph.
This is a structured representation of:
* Files
* Components
* APIs
* DB models
* Dependencies
* Routes
* State
Instead of re-reading raw files each time, it updates the graph.
Example:

Component: Dashboard
  ↳ Uses: useAuth()
  ↳ Calls: /api/users
  ↳ Depends on: UserModel

Now the AI understands impact before making changes and avoids breaking apps.
