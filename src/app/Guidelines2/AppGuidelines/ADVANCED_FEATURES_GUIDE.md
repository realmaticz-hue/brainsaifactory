## 🆘 Troubleshooting

### Avatar Not Generating:
- Check internet connection
- Try different browser
- Clear cache and retry
- Use photo instead of prompt

### Voice Not Working:
- Enable browser speech synthesis
- Check audio permissions
- Try different voice profile
- Restart browser

### Scheduling Fails:
- Verify platform connections
- Check internet connection
- Ensure valid date/time
- Review caption length

### Video Export Issues:
- Reduce video duration
- Lower resolution
- Clear browser cache
- Try different format
