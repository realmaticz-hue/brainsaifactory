## 🎓 Usage Quick Start

### Opening the AI Code Assistant

1. Click the **"AI Code Assistant"** button in the top navigation bar
2. The modal opens with 5 mode tabs: Analyze, Troubleshoot, GitHub, Terminal, Chat

### Analyzing Code (Analyze Mode)

1. Click **"Upload File"** or drag-and-drop a code file
2. Wait for analysis to complete (~3 seconds for 1,000 lines)
3. Review detected issues in the list
4. Click **"Fix All Issues"** for automatic fixing
5. Download or copy the fixed code

### Scanning GitHub Repository (GitHub Scanner Mode)

1. Paste GitHub repository URL (e.g., `https://github.com/username/repo`)
2. (Optional) Add GitHub Personal Access Token for private repos
3. Click **"Scan Repository"**
4. Wait for scan to complete (progress shown)
5. Review issues by file
6. Click **"Fix All Issues"** to auto-fix
7. Download error summary or save fixed files

### Auto-Fixing Terminal Errors (Terminal Mode)

1. Paste your code in the editor
2. Click **"Run Code"**
3. If errors appear, click **"Auto-Fix All Errors"**
4. System iteratively fixes errors (max 10 iterations)
5. View fix summary modal with before/after comparisons
6. Copy or use the fixed code

### Getting Help (Chat Mode)

1. Type your coding question
2. (Optional) Toggle "Include code context" to attach current code
3. Click **"Send"** or press Enter
4. Receive instant AI-powered answers with code examples
