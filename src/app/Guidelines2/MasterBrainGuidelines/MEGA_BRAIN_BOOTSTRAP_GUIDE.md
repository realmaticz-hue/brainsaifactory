## 🎯 Hook System

### Available Hooks

The Mega Brain provides 4 powerful hooks for live development:

| Hook | Purpose | Example |
|------|---------|---------|
| `//GENERATE:<filename>` | Create new file/script/module | `//GENERATE:utils/aiHelper.ts` |
| `//SIMULATE:<function>` | Safely simulate execution | `//SIMULATE:processData` |
| `//BUILD_MODULE:<module_name>` | Auto-create helper AI modules | `//BUILD_MODULE:ErrorAnalyzer` |
| `//VERIFY:<module_or_code>` | Validate correctness and safety | `//VERIFY:utils/aiHelper.ts` |

### How to Use Hooks

#### Method 1: Terminal Input (Recommended)

1. Open the **Terminal** (click "Terminal" button)
2. Type your hook command in the input field at the bottom
3. Press **Enter** to execute

Example workflow:
```bash
$ //GENERATE:utils/aiHelper.ts
🔧 Hook detected: //GENERATE:utils/aiHelper.ts
✅ Generated file: utils/aiHelper.ts
📝 File utils/aiHelper.ts queued for generation
💡 Use //VERIFY to validate the generated code

$ //VERIFY:utils/aiHelper.ts
🔧 Hook detected: //VERIFY:utils/aiHelper.ts
✅ Verified: utils/aiHelper.ts
🔍 Verification checks:
  ✅ syntax
  ✅ security
  ✅ performance
  ✅ bestPractices
```

#### Method 2: During Error Repair

Hooks are automatically detected in any text input within the Git Repair system.

## 🔧 Bootstrap Command Reference

### Command Format

```bash
init-mega-brain [flags]
```

### Available Flags

| Flag | Description | Default |
|------|-------------|---------|
| `--mode=<mode>` | Operation mode (developer-ready, standard, advanced) | standard |
| `--multi-modal` | Enable multi-modal intelligence | off |
| `--self-verifying` | Enable self-verification | off |
| `--tool-building` | Enable tool generation | off |
| `--code-generation` | Enable live code generation | off |
| `--ultra-builder` | Enable ultra-builder mode | off |
| `--meta-cognition` | Enable meta-cognition | off |
| `--safe-mode` | Enable safety & ethics enforcement | off |
| `--personalization` | Enable adaptive personalization | off |
| `--persistent-memory` | Enable persistent memory | off |
| `--hooks=<list>` | Comma-separated list of hooks to enable | none |
| `--auto-extend-subagents` | Enable automatic subagent creation | off |
| `--continuous-self-optimization` | Enable continuous improvement | off |
| `--ethics-alignment` | Enable ethics alignment | off |

### Example Configurations

#### Minimal Configuration
```bash
init-mega-brain --mode=standard --code-generation
```

#### Developer Configuration

## ⚠️ Troubleshooting

### Mega Brain Won't Activate
- Ensure bootstrap command syntax is correct
- Check for typos in flag names
- Refresh page and try again

### Hooks Not Working
- Verify Mega Brain is activated (check for "ACTIVE" badge)
- Ensure hook syntax is correct: `//HOOKNAME:target`
- Open Terminal to see detailed execution logs

### Verification Failures
- Review verification checks in terminal output
- Fix identified issues
- Re-run verification
