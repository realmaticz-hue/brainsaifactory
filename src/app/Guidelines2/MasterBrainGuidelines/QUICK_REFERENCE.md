# Quick Reference Guide

## 🎨 **Professional AI App Builder**

### **Access:**
Click **"App Builder"** button in top navigation (purple button with hammer icon)

### **Quick Start:**
1. Enter app description in text area
2. Click "Generate Complete App"
3. Wait 10-20 seconds
4. Browse through tabs: Preview → Code → Database → API → Export

### **Best Prompts:**
```
✅ "Build a [type] app with [feature 1], [feature 2], and [feature 3]"
✅ Include: user types, features, integrations, special requirements
✅ Be specific about what you want
```

### **Tabs Explained:**

**Preview Tab:**
- Switch between Mobile/Tablet/Desktop views
- Navigate through generated screens
- See your app come to life

**Code Tab:**
- Choose: React, Swift, or Kotlin
- Copy complete source code
- Ready to use immediately

**Database Tab:**
- View all tables and fields
- See relationships
- Understand data structure

**API Tab:**
- All CRUD endpoints listed
- GET, POST, PUT, DELETE methods
- Ready for integration

**Export Tab:**
- ZIP: Complete project files
- XCode: iOS/Mac project
- Android Studio: Android project
- GitHub: Push directly (coming soon)

### **Common App Types:**

| Type | Auto-Generates |
|------|----------------|
| **E-Commerce** | Products, Orders, Cart, Checkout |
| **Social Media** | Posts, Comments, Likes, Followers |
| **Productivity** | Tasks, Projects, Teams, Deadlines |
| **Health/Fitness** | Workouts, Meals, Progress, Goals |
| **Finance** | Transactions, Budgets, Categories |
| **Education** | Courses, Lessons, Progress, Quizzes |

## 🎭 **Professional 3D Avatar Generator**

### **Access:**
Click **"3D Avatar Generator"** button in top navigation (green button with wand icon)

### **Quick Start:**

**Option 1: Text Description**
1. Select "Text" mode
2. Describe your avatar
3. Click "Generate Avatar"
4. Customize in next tabs

**Option 2: Photo Upload**
1. Select "Photo" mode
2. Upload face photo
3. Click "Generate from Photo"
4. Fine-tune the result

**Option 3: Preset**
1. Select "Preset" mode
2. Choose from 6 professional avatars
3. Customize as needed

### **Customization Tabs:**

**Customize Tab:**
- Age slider (18-80 years)
- Gender (Male/Female/Neutral)
- Skin tone color picker
- Eye size, color, shape
- Hair style, color, length
- Smile intensity
- Outfit type and color

**Voice Tab:**
- Voice type (Warm, Professional, Energetic, Calm, Authoritative)
- Accent (American, British, Australian, Indian, etc.)
- Pitch slider (0-100%)
- Speed slider (0-100%)
- Test voice button

**Animate Tab:**
- Default expression (Neutral, Happy, Professional, Friendly, Confident)
- Emotion intensity slider
- Enable/disable head movements
- Enable/disable hand gestures
- Preview animation

**Export Tab:**
- MP4 video (for ads)
- FBX 3D model (for Unity/Unreal)
- GLB 3D model (for web)
- JSON config (to save/reload)

## ⚡ **Pro Tips**

### **App Builder:**
1. **Be Specific**: More details = better results
2. **Think Features**: List all features you want
3. **Consider Users**: Mention user types (admin, customer, etc.)
4. **Preview First**: Check all screens before exporting
5. **Iterate**: Generate multiple versions, pick the best

### **Avatar Generator:**
1. **Match Brand**: Choose avatar that fits your brand
2. **Test Voice**: Always preview voice before finalizing
3. **Check Expressions**: Preview animations to ensure natural look
4. **Consider Audience**: Dress avatar appropriately for viewers
5. **Save Configs**: Export JSON to reuse successful avatars

## 🐛 **Troubleshooting**

### **App Builder:**

**Issue:** Generated app doesn't match description
- **Fix:** Add more specific details to prompt
- **Fix:** Mention exact features needed
- **Fix:** Try regenerating with clearer description

**Issue:** Want to change generated code
- **Fix:** Copy code from Code tab
- **Fix:** Edit in your IDE
- **Fix:** Or regenerate with modified prompt

### **Avatar Generator:**

**Issue:** Avatar doesn't look right
- **Fix:** Use Customize tab sliders
- **Fix:** Try Photo upload instead of text
- **Fix:** Start with preset, then customize

**Issue:** Voice doesn't match avatar
- **Fix:** Change voice type in Voice tab
- **Fix:** Adjust pitch and speed
- **Fix:** Try different accent
- **Fix:** Test voice before finalizing

**Issue:** Animation looks unnatural
- **Fix:** Reduce emotion intensity
- **Fix:** Adjust blink rate
- **Fix:** Toggle head movements off/on
