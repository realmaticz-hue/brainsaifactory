## 🎯 Hook Commands

| Hook | Usage | Purpose |
|------|-------|---------|
| `//GENERATE:file.ts` | Create new file/module | Live code generation |
| `//VERIFY:file.ts` | Validate code | Safety & correctness check |
| `//SIMULATE:funcName` | Test function safely | Safe execution sandbox |
| `//BUILD_MODULE:Name` | Create AI module | Auto-build helper modules |

**How to Use**: Open Terminal → Type hook command → Press Enter
